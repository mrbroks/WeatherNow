# WeatherNow
This is a lightweight weather application built entirely with Python, utilizing the Tkinter library for the graphical user interface (GUI). The app allows users to quickly retrieve basic weather information with minimal dependencies, making it simple and efficient.

### How It Works
The user inputs a city name, and the app sends a request to a weather API. The API responds with weather data in JSON format, from which the app extracts and displays key parameters such as temperature, humidity, and weather conditions.

### Features
- Built with Python and Tkinter for a simple and intuitive GUI.
- Fetches real-time weather data from an API.
- Minimal dependencies for quick setup and usage.

### Project Background
This project was developed for college as part of structured programming course projects. It demonstrates the practical application of API integration, JSON parsing, and GUI design in Python.
